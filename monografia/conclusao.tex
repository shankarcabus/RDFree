\chapter{Considerações Finais}
\label{cap:conclusao}

Este trabalho apresentou o desenvolvimento de uma biblioteca de efeitos sonoros \textit{Open Source} para a plataforma iOS, denominada iSoundFX. O intuito desse trabalho foi oferecer aos desenvolvedores uma alternativa na construção de aplicações musicais, fornecendo uma biblioteca que facilitasse este processo.

Foram integrados à biblioteca os efeitos de \textit{Delay}, Distorção, \textit{Fuzz}, \textit{Overdrive} e \textit{Pitch}. Alguns desses efeitos foram provenientes das aplicações \textit{SkypeFX} e \textit{audioGraph}, e dos trabalhos de \citeonline{chang}, \citeonline{krueger} e \citeonline{harlan}. Foi desenvolvido também um módulo de utilização do \textit{framework} nativo do iOS, com a finalidade de capturar e processar as amostras de áudio. 

Verificou-se na especificação do iSoundFX, que a utilização de aplicações \textit{Open Source} é uma estratégia eficaz, em termos de tempo gasto, na construção de \textit{softwares}. A biblioteca  iSoundFX possibilita o reuso, a legibilidade, e a facilidade de manutenção do código, já que, premissas como a adoção do padrão de projeto \textit{strategy} e a modularização foram empregadas.

Um aplicativo experimental foi desenvolvido com o objetivo de validar o funcionamento do iSoundFX. Os resultados, obtidos com o aplicativo em questão, foram analisados através do espectro dos sinais utilizando o software de edição de áudio \textit{Audacity}. Observou-se, com esta análise, que os algoritmos criados para o iSoundFX atendem as expectativas de funcionamento. 

Como cumprimento de um dos objetivos do trabalho, a biblioteca foi hospedada no \textit{Google Code}\footnote{O projeto está disponível em <http://code.google.com/p/isoundfx/>.} sob a licença \textit{Lesser GNU Public License}, viabilizando, desta forma, que os desenvolvedores possam utilizar o iSoundFX na sua aplicação, sem que haja um valor agregado. 

Como trabalhos futuros dá-se as sugestões:

\begin{itemize}
\item Portar a biblioteca para outras plataformas móveis como \textit{Android} e \textit{Windows Phone} - Considerando que a plataforma dê suporte a captura e execução de áudio em tempo real;
\item Inclusão de novos efeitos - Visto que existe uma quantidade significativa de efeitos sonoros que poderiam ser adicionados ao iSoundFX;
\item Proposta de interface de usuário utilizando a biblioteca desenvolvida - Observou-se que a utilização de componentes de interface de usuário otimizaria o tempo de desenvolvimento das aplicações musicais;
\item Suporte as configurações do módulo de captura: taxa de amostragem, latência de áudio e formato das amostras - Os desenvolvedores não se limitariam às configurações utilizadas pelo iSoundFX. 
\end{itemize}
